# KYBER Plugin Samples

Example Plugins for KYBER.

> [!WARNING]
> KYBER Plugins are still in development and may not work as expected. There is currently no official support or documentation for using plugins. Use at your own risk.

## Plugins

| Plugin                                     | Description |
|--------------------------------------------| ----------- |
| [BattlefieldCamera](BattlefieldCamera)     | Recreates the effect from Battlefield games where your camera smoothly moves towards your spawn position. |
| [BotBalancer](BotBalancer)                 | Balances in-game bot counts according to the gamemode's max players and a desired game density. |
| [OfficialServerTools](OfficialServerTools) | Contains utilities for our official dedicated servers. |
| [GunGame](GunGame)                         | A classic Gun Game mode where players progress through a series of weapons by getting kills. |
| [HVVPlaygroundPlugin](HVVPlaygroundPlugin) | A simple plugin that allows you to voteban players or swap teams using a chat command.
